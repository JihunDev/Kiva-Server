# Kiva-Server

물류로봇을 움직이기우해 모바일 App에서 위치와 물건의 정보를 보여주기 위한 서버



## About the Project

- 데이터베이스에 물건을 저장하기 위한 로직
- 모바일에서 물류로봇으로 위치를 전송
- Json 형태로 모바일 데이터 전송


## Project status

개발 완료



## Getting Started

### Requirements

- Linux

- Node.js

- Socket.io

- MariaDB

- MS Azure
